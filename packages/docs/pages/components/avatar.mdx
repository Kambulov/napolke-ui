## Avatar

Это элемент пользовательского интерфейса, используемый для отображения изображения
профиля пользователя или иконки, которая представляет его. Этот компонент
представляет из себя круглое или квадратное изображение с возможностью добавления обводки или фона, чтобы выделить его на странице.

| Атрибут      | Описание                           | Тип                 | Значение                            | Деф. значение|
| ------------ | ---------------------------------- | ------------------- | ------------------------------------| ------------ |
| **src**      | image src                          | `string`            | -                                   | -            |
| **stacked**  | stacked display group              | `boolean`           | -                                   | `false`      |
| **text**     | display text when image is missing | `string`            | -                                   | -            |
| **isSquare** | avatar shape                       | `boolean`           | -                                   | `false`      |
| ...          | native props                       | `ImgHTMLAttributes` | `'alt', 'className', ...`           | -            |
