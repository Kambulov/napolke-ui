## Button Group

Данный компонент позволяет группировать кнопки

<Playground
  title="Дефолтный вид"
  desc="Базовое отображение `ButtonGroup`  без пропсов"
  scope={{ Button, ButtonGroup, Grid }}
  code={`
<Grid.Container gap={2}>
  <Grid>
    <ButtonGroup>
      <Button>Отправить</Button>
      <Button>Редактировать</Button>
      <Button>Удалить</Button>
    </ButtonGroup>
  </Grid>
</Grid.Container>
`}
/>

<Playground
  title="Типы"
  desc="Установите тип или стили всех кнопок в группе."
  scope={{ Button, ButtonGroup, Grid }}
  code={`
<Grid.Container gap={2}>
  <Grid>
    <ButtonGroup type="success">
      <Button>Отправить</Button>
      <Button>Редактировать</Button>
      <Button>Удалить</Button>
    </ButtonGroup>
  </Grid>
  <Grid>
    <ButtonGroup type="abort">
      <Button>Добавить</Button>
      <Button>Удалить</Button>
    </ButtonGroup>
  </Grid>
  <Grid>
    <ButtonGroup type="warning" ghost>
      <Button>Добавить</Button>
      <Button>Удалить</Button>
    </ButtonGroup>
  </Grid>
  <Grid>
    <ButtonGroup type="success-light">
      <Button>Отправить</Button>
      <Button>Редактировать</Button>
      <Button>Удалить</Button>
    </ButtonGroup>
  </Grid>
</Grid.Container>
`}
/>

<Attributes edit="/pages/components/button-group.mdx">
<Attributes.Title>ButtonGroup.Props</Attributes.Title>

| Attribute    | Description                 | Type                   | Accepted values             | Default   |
| ------------ | --------------------------- | ---------------------- | --------------------------- | --------- |
| **type**     | Тип кнопки                  | `ButtonTypes`          | [ButtonTypes](#buttontypes) | `default` |
| **ghost**    | Прозрачность                | `boolean`              | -                           | `false`   |
| **vertical** | Вертикальное отображение    | `boolean`              | -                           | `false`   |
| **disabled** | Заблокировать кнопки        | `boolean`              | -                           | `false`   |
| ...          | Остальные пропсы            | `ButtonHTMLAttributes` | `'id', 'className', ...`    | -         |

<Attributes.Title>ButtonTypes</Attributes.Title>

```ts
type ButtonTypes =
  | 'default'
  | 'secondary'
  | 'success'
  | 'warning'
  | 'error'
  | 'abort'
  | 'secondary-light'
  | 'success-light'
  | 'warning-light'
  | 'error-light'
```

</Attributes>
