## useBodyScroll

Данный хук позволяет отключать скролл для всей страницы или любого элемента, это полезно для отображения всплывающего элемента или меню.
Если ширина полосы прокрутки страницы установлена на определенное пользователем значение width перед отключением прокрутки, после отключения
будет добавлено значение paddingRight, чтобы адаптироваться.

<Playground
  desc="Click button to disable scrolling."
  scope={{ Button, Spacer, Text, useBodyScroll }}
  code={`
() => {
  const [hidden, setHidden] = useBodyScroll()
  return (
    <>
      <Text small b type={hidden ? 'error' : 'default'}>{hidden ? 'Body заблоикрован.' : 'Body можно прокручивать'}</Text>
      <Spacer h={.75} />
      <Button scale={0.5} type="success-light" auto onClick={() => setHidden(true)}>Блокировать</Button>
      <Button ml="20px" scale={0.5} type="secondary-light" auto onClick={() => setHidden(false)}>Сбросить</Button>
    </>
  )
}
`}
/>
