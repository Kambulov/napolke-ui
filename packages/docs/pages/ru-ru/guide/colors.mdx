## Цвета

Цветовые маркеры помогают поддерживать согласованное использование цвета во всем приложении.
Shoelace предоставляет палитры для тематических цветов и примитивов, которые вы можете использовать в качестве основы для своей системы дизайна.

Ссылки на цветовые маркеры выполняются с помощью пользовательского
свойства --sl-color-{name}-{n} CSS, где {name} - это название палитры, а {n} - числовое значение желаемого образца.
