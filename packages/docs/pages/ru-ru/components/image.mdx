<Attributes edit="/pages/en-us/components/image.mdx">
<Attributes.Title>Image.Props</Attributes.Title>

	| Атрибут       | Описание                                  | Тип                                             | Принимаемые значения        | Деф. значение   |
| --------------------- | --------------------------------------- | ------------------- | ------------------------- | ------- |
| **src**               | путь до файла                              | `string`            | -                         | -       |
| **disableSkeleton**   | отключить анимацию скелета при загрузке | `boolean`           | -                         | `false` |
| **disableAutoResize** | отключить масштабирование, если ширина слишком мала  | `boolean`           | -                         | `false` |
| **maxDelay**(ms)      | максимальная продолжительность анимации              | `number`            | -                         | `3000`  |
| ...                   | стандартные пропсы                          | `ImgHTMLAttributes` | `'alt', 'className', ...` | -       |

<Attributes.Title>Image.Browser.Props</Attributes.Title>

	| Атрибут       | Описание                                  | Тип                                             | Принимаемые значения        | Деф. значение   |
| ---------------- | ------------------------------------- | ---------------------- | ------------------------ | ------- |
| **title**        | показывать текстовый заголовок (если "url" не задан) | `string`               | -                        | -       |
| **url**          | показыват URL-адрес в заголовке     | `string`               | -                        | -       |
| **showFullLink** | показать полный URL-адрес                       | `boolean`              | -                        | `false` |
| **invert**       | инвертировать цвета                        | `boolean`              | -                        | `false` |
| **anchorProps**  | props элемента `a`                  | `AnchorHTMLAttributes` | -                        | `{}`    |
| ...              | стандартные пропсы                          | `HTMLAttributes`       | `'id', 'className', ...` | -       |

</Attributes>
